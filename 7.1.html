class

记住首字母要大写class Animal:我们可以定义类的属性和方法：

class Animal:
    number=3
    leg=4
    def action(self):
        print(self.leg)
        print("action is eat")
    def price(self,buy,sold):
        res=sold-buy
        print('可以赚取金额为:',res)self是必要的一个参数，且第一个参数永远是实例变量self。但是在传递时不必传self。现在就定义好了一个类，我们接下来实例化这个类
                >>>cat=Animal()     # 记住加括号
>>>cat.number
3
>>>cat.action()
4
action is eat
>>>cat.price(10,20)
可以赚取金额为:10

这里应该还有个__init__没有讲，将在下一节继续。
