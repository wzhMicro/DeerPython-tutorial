wxpy

wxpy 在 itchat 的基础上，通过大量接口优化提升了模块的易用性，并进行丰富的功能扩展

安装

和其他模块一样，还是靠pip来安装。pip install wxpy.也许要更加python版本改为pip3

登录

wxpy使用起来非常简单，比如登录开始介绍

from wxpy import *
bot = Bot()       #初始化，扫码登录。
那我们现在开始给名为‘123木头人’的人发送消息 
target = bot.friends().search('123木头人')[0]
target.send('Hello, world!')
这样只要运行脚本就可以自动发送‘Hello world！’给123木头人了，当然这里还可以发送图片视频，名片等资源，并且我们不单单可以 给某个朋友发送，还可以给群聊发送或者公众号发送。这里我们就简单的介绍一下wxpy，后面实际操作几个例子。

  • 好友性别比重分析
  • 好友信息写入csv文件并生成词云
