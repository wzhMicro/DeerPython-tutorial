调用函数

我们上一节已经学会了如何定义一个函数，那现在我们就来学习如何调用我们的函数。

这是上一节的代码。

def func():
    print('Hello World!')
    print('你好，世界！')

如果我们相要调用很简单，只要在脚本内写一遍函数名就可以了。

>>> func()
Hello World!
你好，世界！

return

return 语句就是将结果返回到调用的地方，比如：

def func(a,b):
    x=a+b
    y=a-b
    return y
这段代码，func这个函数计算了2个值，一个是x，一个是y，那么当我们调用这个函数时电脑并不知道 我们需要哪个参数，所以这时候就需要用return来告诉电脑我们真正需要的值是哪一个。这里需要的是y，所以返回y.
