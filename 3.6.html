循环语句

Python中的循环语句有 for 和 while。

我们可以用IDE，脚本或者在cmd下进行Python的交互来进行操作。

for 语句

假设我们现在有一个列表grades=[100,90,99,89,55,98,58].那么我们如果想要打印出所有的成绩就需要用到for了。

for i in grades:
            print(i)

while 语句

while循环会一直循环知道条件不满足

还记得高斯如何计算0~100的和吗，现在我们不用高斯定理也能很快计算出结果。

sum=0
n=100
counter = 1
while counter <= n:
            sum+=counter    #每次循环加上counter值
            counter+=1      #counter自加
print(sum)
得到结果5050

死循环

有时候死循环也会非常有用，还记得上面成绩分类的例子吗，我们输入一个值按下回车，程序就结束了，那我们如何保持程序的运行来持续判断呢。 只要将两个语句结合一下就可以了。

n=1
while n==1:
    grades=int(input("输入一个数字："))

    if grades>=80:
        print("优秀")
    elif 70<=grades<80:
        print("普通")
    elif 60<=grades<70:
        print("及格")
    else:
        print("不及格")

这样我们就可以不用每次都run一次程序了。(如果你在命令提示行下运行可以Ctrl+C来停止程序。如果是IDE中一般都会有停止按钮)
