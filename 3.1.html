变量类型

Python 中的变量不需要声明。每个变量在使用前都必须赋值，变量赋值以后该变量才会被创建。 等号（=）用来给变量赋值。 等号（=）运算符左边是一个变量名,等号（=）运算符右边是存储在变量中的值。

number=0 #整型变量 name="Python" #字符串 score=19.9 # 浮点型变量

多个变量赋值

Python允许你同时为多个变量赋值。例如：

a=b=c=1

或者是：

a,b,c=0,"python",0

String

字符串是以单引号'或双引号"括起来的任意文本，

'This is a string'或者是"This is a string too"

字符串截取是通过[x:y].x是开始截取的起始index，y是截至index。比如：

s="abcdefg";print(s[3:5])s输出结果为：cdef

注意：1.[x:y:z]中第三个参数z代表步长，即按照x，y的方向每次选取z个字符。 2.Python中的字符串有两种索引方式，从左往右以0开始，从右往左以-1开始。

List

列表可以支持数字，字符串甚至是嵌套列表。

定义一个列表是在[ ]中并且以逗号,分隔开。和字符串一样，索引从左往右以0开始，从右往左以-1开始。

具体方法大致如下：

列表可以进行增加或修改，比如：List=['123','234','4455','3434']

尾部增加：List.append('xyz')列表尾部增加一项变为['123','234','4455','3434','xyz']

指定位置增加：List.insert(2,'xyz')列表第二个位置增加一项变为['123','234','xyz','4455','3434']

从列表尾部删除元素：List.pop()列表删除最后一项变为['123','234','4455']

删除指定位置元素：List.pop(2)列表删除第2项变为['123','234','3434']

替换列表元素：List[1]='string'列表第一项被替换['123','string','4455','3434']

Tuple（元组）

元组与列表类似，不同之处在于元组的元素不能修改（定义好后就被定死了，所以较为安全）。元组写在小括号 () 里，元素之间用逗号隔开。

元组的截取方法等和列表一样。值得注意的是当需要定义一个只有一个元素的元组时：

如果写出这样tuple(1)。这样会输出为1这个数字.不为元组。这是因为括号同样也可以表示数学公式中的小括号，所以产生了歧义。

所以需要在最后加上一个逗号来消除歧义。tuple(1,)。这样输出为(1,)

但是有个很有意思的一点: 元组中可以存在列表，处于元组内的列表是可以修改的。tuple=('1','2',['3','4'])可以修改元组第2项['3','4']但是不能更改前两项。

Set（集合）

集合（set）是由一个或数个形态各异的大小整体组成的，构成集合的事物或对象称作元素或是成员。 基本功能是进行成员关系测试和删除重复元素。 可以使用大括号 { } 或者 set() 函数创建集合，注意：创建一个空集合必须用 set() 而不是 { }，因为 { } 是用来创建一个空字典。

Dictionary（字典）

列表是有序的对象集合，字典是无序的对象集合。两者之间的区别在于：字典当中的元素是通过键来存取的，而不是通过偏移存取。 字典是一种映射类型，字典用"{ }"标识，它是一个无序的键(key) : 值(value)对集合。 键(key)必须使用不可变类型。在同一个字典中，键(key)必须是唯一的。

字典的查找是比列表要快的。比如现在有这样两个列表： 

namelist=['max','tom','jane','frank']
IDlist2=['01','02','03','04']
那么如果我们要查找max的ID就需要查找两个列表，这样导致效率不是很高。所以就有了字典的应用：
IDdict={'max':01,'tom':02,'jane':03,'frank':04}
有了字典我们就可以只要查找一遍就可以。
>>>IDdict['max']
01
