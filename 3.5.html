If/Else条件判断

在一个编程语言中，”控制流”(判断语句)是必要的。if – elif – else是Python中的判断语句。

if 条件1：
            语句1
elif 条件2：
            语句2
else：
            语句3

  • 如果条件1为True则执行语句1.
  • 如果条件1为False将判断条件2.
  • 如果条件2为True则执行语句2.
  • 如果条件2为False则执行语句3.

示例

下面我们来看一个简单的例子来了解if – elif – else。假设成绩在80以上为优秀，70~79为普通，60~69为及格,其余为不及格。我们来判断学生成绩的等级。

grades=int(input("输入一个数字："))

if grades>=80:
                print("优秀")
elif 70<=grades<80:
                print("普通")
elif 60<=grades<70:
                print("及格")
else:
                print("不及格")

这样我们就完成了简单的判断语句，你也可以更改程序将之变成一个猜数字的小游戏。动手试试吧。

注意事项

每个条件后面要使用冒号 :，表示接下来是满足条件后要执行的语句块。

Python是很对缩进敏感的，所以要注意编写的逻辑。

if 语句还可以简写，比如：

>>>if x:
>>>    print("True")
只要x是非零数值、非空字符串、非空list等，就判断为True，否则为False.
