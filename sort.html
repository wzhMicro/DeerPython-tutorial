插入排序

原理

每次排序都挑出来一个数，将其放入相应的位置，最后得出就是有序的数列。时间复杂度为O(n^2)，是稳定的排序方法（后面会说稳定是什么意思。

代码

def Insert(list):
n=len(list)
for i in range(1,n):
    j=i
    while j>0 and list[j]>list[j-1]:
        list[j],list[j-1]=list[j-1],list[j]
        j-=1
return list

解析

例如list=[49,38,65,97,76,13,27,49]
第一次：选出来list[1]=38,变换后为[(38,49),65,97,76,13,27,49]
第二次：选出来list[2]=65,变换后为[(38,49,65),97,76,13,27,49]
第三次：选出来list[3]=97,变换后为[(38,49,65,97),76,13,27,49]
第四次：选出来list[4]=76,变换后为[(38,49,65,76,97),13,27,49]
第五次：选出来list[5]=13,变换后为[(13,38,49,65,97,76),27,49]
第六次：选出来list[6]=27,变换后为[(13,27,38,49,65,97,76),49]
第七次：选出来list[7]=49,变换后为[(13,27,38,49,49,65,97,76)]
       到这里就全部排序完成。(list中的括号是方便观看并不是真的创建了个元组)
再解释一下什么是稳定的排序。上面的例子中有两个49，排序前加粗的49在后面，排序后依然在后面， 这样的情况就为稳定的排序。

选择排序

原理

取最小或最大的与第一个交换.时间复杂度是O(n^2),选择排序是不稳定的排序。

代码

def Select(list):
    n=len(list)
    for i in range(0,n-1):
            min_index=i
        for j in range(i+1,n):
            if list[min_index]>list[j]:
                min_index=j
        if i!=min_index:
            list[i],list[min_index]=list[min_index],list[i]
    return list

解析

例如list=[3，1，5，7，2，4，9，6]
第一次：选出来key=1,变换后为[1，3，5，7，2，4，9，6]
第二次：选出来key=2,变换后为[1，2，5，7，3，4，9，6]
第三次：选出来key=3,变换后为[1，2，3，7，5，4，9，6]
第四次：选出来key=4,变换后为[1，2，3，4，5，7，9，6]
第五次：选出来key=5,变换后为[1，2，3，4，5，7，9，6]
第六次：选出来key=6,变换后为[1，2，3，4，5，6，9，7]
第七次：选出来key=7,变换后为[1，2，3，4，5，6，7，9]
       排序完成。

冒泡排序

原理

重复地走访过要排序的元素列，依次比较两个相邻的元素，如果他们的顺序（如从大到小、首字母从A到Z）错误就 把他们交换过来。最小或最大的数会像气泡一样'浮'到顶端，所以叫做冒泡排序 。时间复杂度是O(n^2),冒泡排序是稳定的排序。

代码

def Bubble(list):
    n=len(list)
    for j in range(0,n-1):
        for i in range(0,n-j-1):
            if list[i]>list[i+1]:
                tmp=list[i+1]
                list[i+1]=list[i]
                list[i]=tmp
    return list

解析

例如list=[2,1,4,3,8,5]
第一次：2和1作比较,变换后为[1,2,4,3,8,5]
第二次：2和4作比较,保持原状不变[1,2,4,3,8,5]
第三次：4和3作比较,变换后为[1,2,3,4,8,5]
第四次：4和8作比较,保持原状不变[1,2,3,4,8,5]
第五次：8和5作比较,变换后为[1,2,3,4,5,8]
       排序完成。
个人认为冒泡排序比较低效，但是中间交换的那几步比较关键。 冒泡排序可以改进为鸡尾酒排序，大家有兴趣可以了解一下。
