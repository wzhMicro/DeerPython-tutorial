参数的传递

在每次调用函数时可能想用一些自己的变量去执行函数，那这些变量就是参数。

比如我想计算矩形的面积

def Area(L,W):
    area=L*W
    print(area)

>>>Area(3,4)
12
>>>Area(3,9)
27
在这里矩形的长(L)和宽(W)就是这个函数的两个参数，在调用时输入不同的参数就会有不同的结果。

默认参数

调用函数时，如果没有传递参数，则会使用默认参数。以下实例中如果没有传入 age 参数，则使用默认值，比如：

def grade(name, score, class='class 1'):
    print("名字:",name)
    print("成绩:",score)
    print("班级:",class)

>>>grade(name='max',score=90,class='class 2')
名字:max
成绩:90
班级:class 2
>>>grade(name='sam',score=88)
名字:sam
成绩:88
班级:class 1

可变参数(一个*实现)

上面的例子都是一个参数传入了一个值，Python其实允许我们在一个参数传入多个参数。 通常我们可以以List或者Tuple的形式，但是可变参数写起来明显更加简洁。

def func(*number):
    sum=0
    for i in number:
        sum+=i
    print(sum)

这个程序允许我们计算传入的数字之和。

>>>func(10,2)
12
>>>func(10)
10
>>>func()
0

关键字参数(**实现)

可变参数可以让我们传入0或者多个参数，那么关键字参数是可以传入0个或任意个含参数名的参数。关键字是以字典方式实现。

def func(name,**score):
    print('name:', name, 'score:', score)

>>>func(max)
name: max score: {}
>>>func(max,test1=90,test2=79)
name: max score:{'test1':90,'test2':79}

关键字参数使我们的函数变得更加灵活，因为我们有了更多的'选填项'。

注意：可变参数和关键字参数要写到默认参数前面，否则会被一起操作。 

def dunc(*kw, name, grades)        # 这种定义方法是错误的，name和grades会被吞噬
def func(name,grades, *kw)        # 应该改写成这样
