性别比重分析

我们可以通过wxpy来分析自己好友的组成并且生成图表。

首先还是登录from wxpy import *
bot=Bot()
my_friend=bot.friends()
                接下来我们需要创建一个字典来实现存入性别的数值。sexdict = {'male': 0, 'female': 0, 'other': 0}

for friend in my_friend:
    if friend.sex == 1:                   # 1代表男性
        sexdict['male'] += 1
    elif friend.sex == 2:                 # 2代表女性
        sexdict['female'] += 1
    else:
        sexdict['other']+=1
                接下来我们只要打印这个字典就可以得到相应的内容print(sexdict)比如{'male':184,'female':178,'other':12}
