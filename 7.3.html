继承

面向对象中的继承和现实生活中的继承相同，即：子可以继承父的内容

首先我们定义三个类。class Animal:
    def action(self):
        print("Animal is eating!")

class Cat:
    def action(self):
    print("Animal is eating!")

class Dog:
    def action(self):
    print("Animal is eating!")我们可以发现，三个类但是方法都是一样的，那这样写即繁琐又复杂。那就来改写成继承的形式吧！
                class Animal:
    def action(self):
        print("Animal is eating!")

class Cat(Animal):
    pass

class Dog:
    pass这样就好了，我们来试试运行。
                >>>cat=Cat()
>>>cat.action()
Animal is eating!这样功能同样可以实现。这里Animal是Cat和Dog的父类，Cat和Dog是Animal的之类。那我们再将代码改成下面的形式：
                class Animal:
    def action(self):
        print("Animal is eating!")

class Cat(Animal):
    def action(self):
        print("cat is eating!")

>>>cat=Cat()
>>>cat.action()
cat is eating!这就是继承的另一个好处。当子类有自己的函数时是可以将父类的给覆盖了，所以非常灵活。
